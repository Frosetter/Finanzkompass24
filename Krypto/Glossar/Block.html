Block

  • Block: Eine Einheit (eine Art Ordner) in der Blockchain, die eine Gruppe von Transaktionen oder Daten enthält.
  • Header: Enthält Metadaten wie Zeitstempel, den Hash des vorherigen Blocks und den eigenen Hash.
  • Transaktionsliste: Eine vollständige Aufzeichnung aller Transaktionen, die in diesem Block gespeichert sind.
  • Verkettung: Jeder Block ist kryptografisch mit dem vorherigen Block verbunden und sichert so die Integrität der gesamten Kette.
  • Unveränderlichkeit: Nachträgliche Änderungen an einem Block würden die gesamte Blockchain ungültig machen, was Manipulation extrem erschwert.
